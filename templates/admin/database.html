How These Buttons Work
Re-Seed Now — Inserts sample users, projects, materials & transactions into the current database immediately.
Prevent Seeding on Restart — Creates a flag file so the app skips auto-seeding next time it starts. (Only available when flag is absent)
Allow Seeding on Restart — Removes the flag file so sample data loads automatically on the next app restart. (Only available when flag exists)

Danger Zone — Full Database Reset

This will permanently delete ALL data — every user, project, bid, transaction and message. A fresh admin account will be recreated automatically. You will be logged out. This cannot be undone.
